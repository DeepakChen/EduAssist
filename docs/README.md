# 智教助手

智教助手是一个为教育环境设计的电教管理工具，旨在简化学校计算机的日常维护、任务调度和软件管理。该项目特别优化了触屏大屏设备（如希沃、鸿合HiteVison）的使用体验。

## 功能简介
- **定时开关机**：根据学校作息时间自动开关机。
- **定时课程表任务**：根据课程表自动打开相关教学工具。
- **软件静默安装器**：快速部署常用教学软件。
- **系统疑难解答**：一键修复常见的系统问题。
- **触屏优化**：为触屏设备提供更好的操作体验，支持快捷侧边栏。
- **安全工具**：内置基础安全工具，保护计算机免受威胁。
- **云同步**：配置文件和数据同步到GitHub，便于多台设备管理。
